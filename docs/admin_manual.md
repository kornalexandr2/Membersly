# Руководство администратора Membersly

Добро пожаловать в систему автоматизации платного доступа Membersly.

## Начало работы

1.  **Настройка бота:** Убедитесь, что ваш токен от `@BotFather` прописан в настройках сервера (`.env`).
2.  **Подключение каналов:** Добавьте бота в ваш канал или группу в качестве **Администратора** с правами:
    *   Добавление участников (Пригласительные ссылки).
    *   Удаление/Бан пользователей.
3.  **Тарифы:** Создайте тарифы через веб-интерфейс администратора (в разработке). Укажите цену, срок действия и привяжите к нужному каналу.
4.  **Оплата:** Пользователи переходят по ссылке бота, выбирают тариф и оплачивают. После успешной оплаты они автоматически получают доступ.

## Отзыв доступа
Бот автоматически проверяет подписки пользователей. Когда подписка истекает и не может быть продлена, бот удаляет пользователя из закрытого канала/группы.
