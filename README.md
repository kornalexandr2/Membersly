# Membersly

**Membersly** — это полнофункциональное SaaS-решение для полной автоматизации платных подписок в Telegram-каналах и группах.

## Основные возможности

- **Мульти-бот система:** Возможность подключения и управления несколькими Telegram-ботами из одной панели.
- **Управление ресурсами:** Легкое добавление/удаление каналов и групп для автоматизации доступа.
- **Автоматизация доступа:** Бот сам принимает оплату, одобряет заявки на вступление и удаляет пользователей по истечении подписки.
- **Платежные шлюзы:** Поддержка **Telegram Stars** (XTR) и **ЮKassa** (RUB) с рекуррентными платежами.
- **Watchdog система:** Фоновый воркер контролирует статус подписок 24/7.
- **Гибкие тарифы:** Настройка пробных периодов, длительности и уровней доступа.

## Технологический стек

- **Backend:** Python 3.12, FastAPI, Aiogram 3.x, SQLAlchemy 2.0.
- **Frontend:** React, Vite, TailwindCSS, Shadcn/UI.
- **Инфраструктура:** PostgreSQL 16, Redis, Arq, Docker Compose, Nginx.
